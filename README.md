## Building

Install [grunt.js](https://github.com/gruntjs/grunt/blob/0.3-stable/docs/toc.md).

Go to Terminal and `cd` into the billy-data folder and run:

```
grunt
```

## Running tests

Just open the `tests.html` file in your browser.
